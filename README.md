# fake-store
Project for simple API practice for e-commerce.

## Technologies
Project is created with:
* HTML/CSS/Javascript
* React.js
* REST API
* axios
* Redux
* Fuse

## Setup
To run this project, install it locally using npm:

```
$ cd fake-store
$ npm install
$ npm i react
$ npm i react-router-dom
npm i react-redux
$ npm i axios
$ npm install --save fuse.js
$ npm start
```
